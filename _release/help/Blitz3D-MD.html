Blitz3D-MD...

Blitz3D-MD is based on Blitz3D and has made some modifications to it. You can obtain the source code from GitHub

You can view the changes by looking at ChangeLog.txt

Some things about Blitz3D, such as tutorials, examples, etc., can be generalized on Blitz3D-MD

You can use mediaverageexe to open the media file

Please do not use IDE to write or save code. Instead, use code editors such as Visual Studio Code to write it
